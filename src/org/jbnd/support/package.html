Classes that support the JBND library; some of them are essential to the library, some are not. The classes that the programmer at some point might need to interact with directly are:
  • Filter and AbstractFilter - used with org.jbnd.data.FilteringProxyDataSource
  • NamingSupport - for deriving user presentable labels and such from key(path)s
  • Utilities - for various common operations
  • ValidationExceptionSupport - for translating org.jbnd.ValidationExceptions into user feedback
